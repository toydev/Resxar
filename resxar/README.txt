■ 3. アプリ概要
------------------------

数百、数千のテキストや画像などの各種リソースを
１つの Resx 形式ファイルに固めるためのお手軽開発補助ツールです。

詳しくは wiki（https://github.com/toydev/Resxar/wiki）を参照のこと。

■ 4. インストール／アンインストール
------------------------

インストールは Resxar.zip の中身をパスの通ったディレクトリに解凍し配置するだけです。
また、アンインストールもファイルを削除するだけです。
